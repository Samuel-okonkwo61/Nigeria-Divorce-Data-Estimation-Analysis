# Nigeria-Divorce-Data-Estimation-Analysis
Data analysis project estimating divorce trends in Nigeria using demographic and socioeconomic indicators. Built interactive Power BI dashboards to explore regional patterns, marital stability factors, and generate data-driven insights.







	Nigeria Divorce Data Estimation Analysis
Overview
This project analyzes and estimates divorce patterns in Nigeria using available demographic and social indicators. Due to the limited availability of structured national divorce statistics, this project focuses on data estimation, exploratory analysis, and visualization to understand potential trends and factors influencing divorce rates.
The goal of the project is to demonstrate data analysis, data modeling, and insight generation skills using real-world social data challenges.
________________________________________
Problem Statement
Reliable divorce statistics in Nigeria are often incomplete or inconsistently reported, making it difficult to understand the scale and drivers of marital dissolution.
This project addresses the question:
How can available demographic and social indicators be used to estimate and analyze divorce patterns in Nigeria?
________________________________________
Data Sources
The dataset used in this project includes estimated indicators related to marital trends, such as:
•	Population statistics
•	Marriage and divorce estimates
•	Regional demographic patterns
•	Socioeconomic indicators
These datasets were cleaned and structured to support analytical exploration and visualization.
